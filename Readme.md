# 🎧 Spotify Music Analysis Dashboard (Power BI)

## 📊 Overview
This project presents an interactive Power BI dashboard built using Spotify dataset to analyze music trends, artist distribution, and song characteristics.

The dashboard focuses on identifying:
- Artist contribution and dominance  
- Monthly and yearly song trends  
- Explicit vs non-explicit content distribution  
- Popularity patterns across time  

---

## 🎯 Key Insights
- A small number of artists contribute a large portion of total songs, indicating a highly skewed distribution.  
- Song popularity shows seasonal variation, with noticeable dips and recoveries during certain months.  
- Higher content volume does not always result in higher popularity.  
- Majority of songs are distinct, indicating minimal duplication in dataset.  

---

## 📌 Features
- KPI cards for total songs, artists, and explicit content  
- Artist-wise song distribution  
- Monthly trends for popularity and song count  
- Interactive filters for songs and artists  
- Clean and user-friendly UI design
